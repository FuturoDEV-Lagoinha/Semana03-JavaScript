Estruturas de controle de fluxo

Condicional

Se a condição for verdadeira, faça algo.

Se a condição for falsa, faça outra coisa.

Existem 2 formas de escrever esse tipo de condição.

  • IF
    • if (condição) { código }

    • if (condição) { código } else { código }

      • OBS: também podemos escrever com o operador ternário: condição ? código : código
    • if (condição) { código } else if (outra condição) { código } else { código }
  • SWITCH CASE

Repetição

WHILE / DO WHILE

Enquanto a condição for verdadeira, faça algo.

Enquanto a condição for falsa, faça outra coisa.

Existem 2 formas de escrever esse tipo de condição.

  • Forma 1 - WHILE: while (condição) { código }

    Verifica a condição e depois faz

  • Forma 2 - DO WHILE: do { código } while (condição)

    Faz e depois verifica a condição

Área de prática

FOR

Para cada item da lista, faça algo.

Existem 3 formas de escrever esse tipo de condição.

  • Forma 1 - FOR: for (inicialização; condição; incremento) { código }
  • Forma 2 - FOR OF: for (variável of lista) { código }
  • Forma 3 - FOREACH: lista.forEach((variável) => { código })
